### 本地开发

1. 克隆项目到本地

```bash
git clone https://github.com/AheheXx1982/SilentX
```

2. 进入项目目录并安装依赖

```bash
cd cos-space
pnpm i
```

3. 启动项目

```bash
pnpm dev
```

#SilentX 更新文档：
----------------------------------------------------------------------16/8/2025

1.更新 Astro 至 5.13.2 最新版本

# 更新 Astro 核心包

```bash
npm install astro@5.13.2
```

# 或者使用 yarn

```bash
yarn add astro@5.13.2
```

# 更新所有 Astro 相关包

```bash
npm update @astrojs/\* --latest
```

# 添加 React 集成
